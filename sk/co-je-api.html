Čo je API?

API sú mechanizmy, ktoré umožňujú dvom softvérovým komponentom vzájomnú komunikáciu pomocou súboru definícií a protokolov. Napríklad softvérový systém meteorologickej služby obsahuje denné údaje o počasí. Aplikácia počasia vo vašom telefóne „komunikuje“ s týmto systémom prostredníctvom API a zobrazuje vám denné aktualizácie počasia na vašom telefóne.
