Čo je server?

Server je počítač alebo systém, ktorý poskytuje zdroje, dáta, služby alebo programy iným počítačom (klientom) prostredníctvom siete. Existuje mnoho typov serverov vrátane webových, poštových a virtuálnych serverov.

Čo je proxy server?

Server proxy umožňuje klientom nepriame pripojenie k inému serveru, pričom funguje ako sprostredkovateľ, ktorý prekladá požiadavky a chráni identitu klienta.

Čo je reverzný proxy server?

Reverzný proxy server sa klientovi javí ako bežný webový server, ale v skutočnosti preposiela požiadavky na jeden alebo viac koncových serverov za ním. Používa sa na zvýšenie bezpečnosti a výkonu.
