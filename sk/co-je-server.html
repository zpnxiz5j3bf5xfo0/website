Čo je server?

Server je počítač alebo systém, ktorý poskytuje zdroje, dáta, služby alebo programy iným počítačom, známym ako klienti, prostredníctvom siete. Teoreticky sa počítače vždy, keď zdieľajú zdroje s klientskymi počítačmi, považujú za servery. Existuje mnoho typov serverov vrátane webových serverov, poštových serverov a virtuálnych serverov. Jeden systém môže poskytovať zdroje a zároveň ich používať z iného systému. To znamená, že zariadenie môže byť súčasne serverom aj klientom.

Čo je proxy server?

Server proxy alebo proxy server je server počítačovej siete, ktorý umožňuje klientom nepriame pripojenie k inému serveru. Proxy server funguje ako sprostredkovateľ medzi klientom a cieľovým serverom, prekladá požiadavky klienta a oproti cieľovému serveru vystupuje ako klient. Prijatú požiadavku potom odosiela naspäť klientovi. Môže ísť tak o špecializovaný hardvér, ako aj o softvér bežiaci na počítači.

Čo je reverzný proxy server?

Reverzný proxy je proxy server, ktorý sa klientovi javí ako bežný webový server, ale v skutočnosti funguje iba ako sprostredkovateľ, ktorý preposiela požiadavky klienta na jeden alebo viac bežných webových serverov. Medzi bežných poskytovateľov reverzných proxy serverov patria Cloudflare a Akamai. Reverzné proxy servery pomáhajú zvýšiť škálovateľnosť, výkon, odolnosť a bezpečnosť, ale nesú so sebou aj množstvo rizík. Útočníci môžu napríklad niekedy použiť nástroje ako Censys alebo Shodan na zistenie pôvodnej IP adresy a obídenie reverzného proxy servera.
