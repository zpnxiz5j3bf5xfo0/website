Čo je DNS?

Domain Name System (DNS) je telefónny zoznam internetu. Ľudia zvyčajne pristupujú k informáciám online prostredníctvom názvom domén, ako sú napríklad example.com alebo example.org. DNS prekladá tieto názvy domén na IP adresy, aby s nimi mohli internetové zariadenia komunikovať.

Čo je DNS záznam?

DNS záznamy sú pokyny, ktoré žijú na autoritatívnych serveroch DNS a poskytujú informácie o doméne vrátane toho, aká IP adresa je priradená k danej doméne a ako spracovať požiadavky pre túto doménu. Tieto záznamy pozostávajú zo série textových súborov napísaných v syntaxi DNS. Syntax DNS je len reťazec znakov používaných ako príkazy, ktoré hovoria serveru DNS, čo má robiť.

Typy DNS záznamov

A záznam - záznam, ktorý obsahuje IPv4 adresu domény.

AAAA záznam - záznam, ktorý obsahuje IPv6 adresu pre doménu.

CNAME záznam - Preposiela jednu doménu alebo subdoménu do inej domény.

MX záznam - nasmeruje poštu na e-mailový server.

TXT záznam - umožňuje správcovi ukladať do záznamu textové poznámky. Tieto záznamy sa často používajú na zabezpečenie e-mailov.

NS záznam - Ukladá názvový server pre položku DNS.

SOA záznam - Ukladá informácie správcu o doméne.

Ako nájdem IP adresu konkrétnej domény?

Na tento účel môžete použiť príkaz nslookup nasledovaný názvom domény v termináli vášho systému. Ak napríklad chcete zistiť IP adresu google.com, použijete:

nslookup google.com

Výsledok bude vyzerať približne takto:

Server:        67.207.67.3
Address:       67.207.67.3#53

Non-authoritative answer:
Name:	google.com
Address: 142.250.184.238
Name:	google.com
Address: 2a00:1450:4001:82b::200e
Adresy v neautoritatívnej odpovedi sú IP adresy domény.

Čo je nameserver?

Nameserver je typ DNS servera. Je to server, ktorý ukladá všetky DNS záznamy pre doménu, vrátane A, MX, alebo CNAME záznamov.
